# Iron Chests - Fabric Mod
_[Fabric] Iron Chests is a complete Fabric port of the original Iron Chest mod. It adds a variety of new chests to the game with different storage sizes. Each chest can each be upgraded over time as you collect and store more materials!_

# Downloads

*Remember to always backup your worlds before adding or updating mods!*

### Recommended Releases

The latest stable releases can always be found on either [CurseForge](https://www.curseforge.com/minecraft/mc-mods/iron-chests-fabric) or [Modrinth](https://modrinth.com/).<br/>
The top file in the list is the latest recommended release!

[![](http://cf.way2muchnoise.eu/versions/iron-chests-fabric_latest.svg)](https://www.curseforge.com/minecraft/mc-mods/iron-chests-fabric)

### Development Builds

If you are looking for the latest bleeding edge build, you can find unstable releases [here](https://github.com/TechnoVisionDev/IronChest/releases).</br>
Use these builds with caution, and please do not put these in modpacks. They may contain major bugs!

# Issues and Suggestions
To report issue, add a translation, or make a feature suggestion, please open a new issue in the `Issues` tab. This will require that you create a free GitHub account. **Please always include the version of IronChest that you are using!**
